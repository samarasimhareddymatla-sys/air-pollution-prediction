# Air Pollution Prediction using Machine Learning
##Project Overview
This project focuses on **environment monitoring and pollution control** by predicting air pollution levels (PM2.5).  
Using **Machine Learning models (Linear Regression & Random Forest)**, the system analyzes pollution data and predicts PM2.5 concentration.  
This can help in **air quality monitoring** and provide insights for pollution control measures.

## Features
- Data Cleaning and Preprocessing  
- Exploratory Data Analysis (EDA) with visualization  
- Machine Learning models:  
  - Linear Regression  
  - Random Forest Regressor  
- Model performance comparison (MSE & R² Score)  
- Visualization of **Actual vs Predicted PM2.5 values**

##Dataset
- Input: CSV file containing air quality data (with **PM2.5** column).  
- Size reduced to 3000 rows for faster training.  
- Ensure your dataset has a **`PM2.5`** column (target variable).
